用JavaScript自动加载网页中的图片列表

  • javascript
  • 闭包


标注图像的时候，由于图像分辨率很高，加载起来很慢。每点击一张图像，网速差的话要等一分钟，浪费时间。但是已经加载过的图像，因为浏览器有cache，重新加载就很快。
所以我想，可以写程序挨个加载Pending中的图片，都加载好了，我再开始标注，就不用等待啦。


html结构

html的代码结构大概是这样：有两个class=“task-list”的div，分别对应未标注的图像（Pending）、已提交的图像（Done）

2个div下各有一组li，点击后开始加载对应编号的图像。

How to do

我们的任务可以分解为：

  1. 得到pending的div，即class=“task-list”的div
  2. 得到这个div下所有的li
  3. 触发li的点击事件
  4. sleep若干秒，等待图像加载完毕

效果

得到了pending中的19个li

click后，图片在加载中…

我就可以先干别的事了

注意

  • JavaScript是单线程的，一次只能完成一个任务，有多个任务就需要排队
  • Javascript将任务的执行模式分成两种：同步（Synchronous）和异步（Asynchronous）
  • 在浏览器端，耗时很长的操作（click、setTimeout）都为异步执行，避免浏览器失去响应

这对零基础的人不太友好，异步、闭包这样的概念我还不太能理解。
注意要设置允许浏览器缓存。
