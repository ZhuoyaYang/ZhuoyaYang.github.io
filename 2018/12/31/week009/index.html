LeetCode

括号匹配:

  • Valid Parentheses 三种括号[]{}()的匹配。用栈，有一种很巧妙的方法。
  • Longest Valid Parentheses hujx去猿辅导的最后一道题哦。只有一种括号()，求最长可匹配括号长度（substrig）。

Tree:

  • Construct Binary Tree from Preorder and Inorder Traversal 根据前序、中序恢复二叉树，递归。
  • Construct Binary Tree from Preorder and Postorder Traversal 根据中序、后序恢复二叉树，递归。会了上面那个之后，so easy。
  • Maximum Binary Tree 和上面的重建二叉树有点类似的题。每次从数组中找最大的为根，以根为界的左数组生成左子树，以根为界的右数组生成右子树。
  • Path Sum II 这是一道还需要再巩固的题。求root到leaf，和为target的所有路径。
  • Sum Root to Leaf Numbers 和Path Sum II类似，都是根到叶的路径问题。每条root到leaf的路径组成一个整数，求所有整数的和。
  • Binary Tree Zigzag Level Order Traversal Z字形层序遍历，奇数行从左至右；偶数行从右至左。好题，要再做。
  • Find Bottom Left Tree Value 和Z字形遍历类似，都是要一层一层的遍历。求二叉树最后一层的最左侧节点值。
  • Binary Tree Right Side View 从右侧看一棵树，从上至下是哪些节点。dfs。
  • Maximum Width of Binary Tree 不错的题，树的宽度的变形。bfs，用到完全二叉树的性质。
  • Find Largest Value in Each Tree Row 找到每行最大的元素。dfs，对比从右侧看一棵树那道题，注意遍历顺序。
  • Flip Equivalent Binary Trees 一棵树可否反转一些节点的左右子树，变成另一棵树。
  • Binary Tree Inorder Traversal 复习，非递归中序遍历，居然忘记了，😢。
  • Binary Tree Preorder Traversal 非递归前序遍历。
  • Kth Smallest Element in a BST 在二叉搜索树中找第K小的，我按非递归中序遍历做的，可能有更好的方法。
  • All Nodes Distance K in Binary Tree 不错的题。找离target距离为K的节点。
  • Add One Row to Tree 加入树的第d层，值为v。原来的左子树变成新加入左子树的左子树。有点无聊的一道题。
  • Diameter of Binary Tree 复习，求树的直径，居然也忘记了，😢。

others:

  • Generate Parentheses 有多少组匹配的括号序列，dfs。
  • Search a 2D Matrix II 一个二维数组，每行/列递增，找目标数。
  • Task Scheduler 任务最小完成时间，相同任务直接需要间隔n个CPU周期（可以做别的任务）。
  • Group Anagrams 错位词分组。
  • Find All Anagrams in a String 从p中找s的所有错位词的起始index。
  • Unique Paths m*n的网格，从左上角到右下角的不同路径有多少条。基础dp。
  • Subarray Sum Equals K 求数组中连续subarray之和等于target的subarray个数。用了unordered_map，非常精妙的解法。
  • Find All Duplicates in an Array 有一个数组，所有元素值在1-n(array.size)之间。有一些数出现了2次，找出这些出现了2次的数。要求时间O(n)， 空间O(1)。
  • Sort Colors 将只含有0、1、2的数组，原地排序。两种解法，计数排序，双指针。双指针看了好久不会。
