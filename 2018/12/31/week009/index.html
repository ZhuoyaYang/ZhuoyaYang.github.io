LeetCode

Tree:

  • Construct Binary Tree from Preorder and Inorder Traversal 根据前序、中序恢复二叉树，递归。
  • Maximum Binary Tree 和上面的重建二叉树有点类似的题。每次从数组中找最大的为根，以根为界的左数组生成左子树，以根为界的右数组生成右子树。
    others:
  • Generate Parentheses 有多少组匹配的括号序列，dfs。
  • Search a 2D Matrix II 一个二维数组，每行/列递增，找目标数。
  • Task Scheduler 任务最小完成时间，相同任务直接需要间隔n个CPU周期（可以做别的任务）。
  • Group Anagrams 错位词分组。
  • Find All Anagrams in a String 从p中找s的所有错位词的起始index。
  • Unique Paths m*n的网格，从左上角到右下角的不同路径有多少条。基础dp。
  • Subarray Sum Equals K 求数组中连续subarray之和等于target的subarray个数。用了unordered_map，非常精妙的解法。
  • Find All Duplicates in an Array 有一个数组，所有元素值在1-n(array.size)之间。有一些数出现了2次，找出这些出现了2次的数。要求时间O(n)， 空间O(1)。
  • Sort Colors 将只含有0、1、2的数组，原地排序。两种解法，计数排序，双指针。双指针看了好久不会。
