LeetCode(650) 2 Keys Keyboard

题目
最开始有1个A，每次可以copy现有的字符串，或者paste上次copy的内容，问得到n个A最少需要几步。

思路
用dp[i]表示得到i个A最少需要几步。

  1. 边界条件：
    dp[1]=0，dp[2]=2（1次copy1次paste），dp[3]=3（1次copy2次paste）….
  2. 转移方程：
    当i>j，i%j==0，dp[i]可以由dp[j]转移过来。
  • 由最大的dp[j]转移过来step最小。比如得到100个A，显然由50个A复制粘贴最快，而不是25个
  • j最大=i/2

一些一开始没有考虑到的地方
这道题比较好想，很快就过了。但时间复杂度为O(n*n)，65ms，只打败了19%。说明有地方没有考虑到，一些判断或循环可以省略。cnblog上一篇文章提出了一些优化方法。

LeetCode(279) Perfect Squares

n可以用最少多少个完全平方数表示。

思路
暴力+普通dp+四平方和定理
方法一:
n可以进行拆解：
dp(0) = 0;
dp(1) = 1;
dp(2) = dp[2-1*1]+1;
dp(3) = dp[3-1*1]+1;
dp(4) = min(dp[4-2*2]+1,dp[4-1*1]+1)
dp(5) = min(dp[5-2*2]+1,dp[5-1*1]+1)
dp(10) = min(dp[10-3*3]+1,dp[10-2*2]+1,dp[10-1*1]+1)

LeetCode(357) Count Numbers With Unique Digits

题目
对于0 ≤ x < pow(10,n)，有多少个x每位数字都不重复。

思路
这是一道数学题，不是非用dp不可。

  • n=0时，有1个，0
  • n=1时，有10个，0-9
  • n=2时，十位数可以从1-9上选择，个位数可以从0-9中不同于十位数的剩下9个数中选择，有 9 * 9 个
  • n=3时，百位及十位可以沿用n=2的时候的81种，个位可以从0-9中与百位十位不同的剩下8个数中选择，有 81 * 8 = 648 个
  • n=4时，有 648 7 = 4536 个
    以此类推，对于一个n位数（n>=2），不重复的n位数的个数 = 不重复的n-1位数的个数    （11-n）

一个很简洁的写法
