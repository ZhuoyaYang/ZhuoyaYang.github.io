LeetCode(494) Target Sum

题目
题目的大致意思是给定一组非负数，在数的前面任意加上“+”或者“-”，让最后结果等于s，问有多少种可能。
题目中提示，数组的最大长度为20，初始数组的和不大于1000，输出结果在int范围内。
Example 1:
Input: nums is [1, 1, 1, 1, 1], S is 3.
Output: 5
Explanation:
-1+1+1+1+1 = 3
+1-1+1+1+1 = 3
+1+1-1+1+1 = 3
+1+1+1-1+1 = 3
+1+1+1+1-1 = 3
思路
用dp[i][j]表示在第i个数字前添加符号后，sum=j的方法数量。
因为要使sum为非负数，所以加上一个bias=1000。
边界条件
边界条件为：当只有一个数时，dp[0][val+1000]++;dp[0][-val+1000]++;
注意dp[0][val+1000]=1;dp[0][-val+1000]=1;是错的。
当val=0是，显然dp[0][1000]应该为2。

LeetCode(516) Longest Palindromic Subsequence

题目
求一个字符串的最长回文子序列长度。
Note:首先要区分Subsequence（子序列）和Substring（子串）。这里是Subsequence，可以是不连续的。
思路
解法一：将字符串reverse，求与原字符串的最长公共子序列（LCS），即为最长回文子序列。
解法二：
对于任意字符串：

  • 如果头尾字符相同，字符串的最长子序列等于去掉首尾的字符串的最长子序列加上首尾；
  • 如果首尾字符不同，则最长回文子序列等于 max{去掉头的字符串的最长回文子序列,去掉尾的字符串的最长子回文序列}。
    用dp[i][j]表示字符串s[i…j]的最长回文子序列长度。
    边界条件 转移方程 关于i,j两层循环的顺序问题、递增递减问题：
  • j>=i+1，j的大小依赖i,所以j的循环嵌套在i内；
  • dp[i][j]的大小为dp[i+1][*]、dp[*][j-1]传递而来，i应该由大至小，j应该由小至大。

LeetCode(213) House Robber ii

题目
这道题是之前House Robber的拓展，现在房子排成了一个圆圈，因为收尾相连，第一家和最后一家只能抢一个。
可以变通一下：把第一家和最后一家分别去掉，各算一遍能抢的最大值，然后比较两个值，较大的一个即为所求。
Example 1:
Input: [2,3,2]
Output: 3
Explanation: You cannot rob house 1 (money = 2) and then rob house 3 (money = 2),because they are adjacent houses.
