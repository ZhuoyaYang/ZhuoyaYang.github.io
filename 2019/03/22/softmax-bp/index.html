Assignment-1 Softmax的实现

发表于 2019-03-22


前向传播计算softmax的损失、反向传播更新参数。详情请看作业要求。
发现网上的答案都很简单，关于W的梯度没有详细推导。所以把我推的顺手搬上来。

参数

Input

  • W: A numpy array of shape (D, C) containing weights.
  • X: A numpy array of shape (N, D) containing a minibatch of data.
  • y: A numpy array of shape (N,) containing training labels; y[i] = c means that X[i] has label c, where 0 <= c < C.
  • reg: (float) regularization strength

Output

  • loss, dW

前向传播计算loss,dW

  • softmax层的输入为X，维度为（N，D）。表示输入batchsize=N个样本，每个样本有D维特征。
    • $X_{i}$表示一个样本的D维特征
  • weights为W，维度为（D，C）。
    • $W_{j}$表示某个样本的D维特征，分别对类别$j$的贡献程度

with loops

要求使用循环，逐个样本计算损失$L_{i}$，逐个样本逐个类别计算梯度$\partial W_{j}$。

Loss的计算

$X_{i} W$，得到$X_{i}$大小为（1,C）的得分向量，记为$S$。
$X_{i} W_{j}$，得到$X_{i}$对于类别$j$的得分，记为$S_{j}$

样本$i$的loss为: $L_{i} = -log(\frac{e^{S_{y_{i}}}}{\sum_{j=1}^C e^{S_{j}}}) = -S_{y_{i}}+log(\sum_{j=1}^C e^{S_{j}}) = -X_{i}W_{y_{i}}+log(\sum_{j=1}^C e^{X_{i}W_{j}}) $

加上l2正则化约束后： $Loss = \sum_{i}^N L_{i} + \frac{reg}{2}{||W||_{2}}^2$

$\frac{\partial L_{i}}{\partial W_{j}}$的计算

当$j = y_{i}$，$\frac{\partial L_{i}}{\partial W_{j}} = -X_{i} + \frac{X_{i}e^{X_{i}W_{j}}}{\sum_{k=1}^C e^{X_{i}W_{k}}}$
当$j \ne y_{i}$，$\frac{\partial L_{i}}{\partial W_{j}} = \frac{X_{i}e^{X_{i}W_{j}}}{\sum_{k=1}^C e^{X_{i}W_{k}}}$

我们知道第$i$个样本softmax之后$j$类的概率：$Softmax_{i,j} = \frac{e^{X_{i}W_{j}}}{\sum_{k=1}^C e^{X_{i}W_{k}}}$
所以$\frac{\partial L_{i}}{\partial W_{j}}$可以简化为：

当$j = y_{i}$，$\frac{\partial L_{i}}{\partial W_{j}} = X_{i}Softmax_{j} - X_{i} = X_{i}(Softmax_{j}-1)$
当$j \ne y_{i}$，$\frac{\partial L_{i}}{\partial W_{j}} = X_{i}Softmax_{j}$

vectorized version

同理with loops，只是找到规律后，可以不用循环得到结果。
$Loss$很简单，不说了。

当$j = y_{i}$ , $\frac{\partial Loss}{\partial W} = X’\left[Softmax-1\right]$
当$j \ne y_{i}$ , $\frac{\partial Loss}{\partial W} = X’Softmax$
