准备KITTI数据集

浏览器官网下载

在kitti官网下载，在导航栏选择object-3d。选择要下载的数据集，填写邮箱，官网会发送一个下载链接给你。
根据README，需要的数据集文件夹组织如下：

  • image_2：对应官网left color images of object data set (12 GB)
  • calib：对应官网camera calibration matrices of object data set (16 MB)
  • label_2：对应官网training labels of object data set (5 MB)
  • velodyne：对应官网Velodyne point clouds, if you want to use laser information (29 GB)
  • velodyne_reduced：自己创建空文件夹即可

浏览器下载比较慢，29G的文件有时候会下载到一半失败。而且官网链接，用wget下载，即使设置--user-agent伪装成浏览器，也下载不成功。所以下面，找了一个私链，伪装浏览器+断点重传，可成功下载，十分方便。

wget私链下载

私链，查看要下载的数据集的链接。

4个zip中除了data_object_label_2.zip只有training，其他3个都包含training和testing。直接解压，再建立velodyne_reduced，刚好就是README中的组织结构。
29G的velodyne数据集1-2M/s大概要下载6个小时，用nohup wget -c --user-agent="Mozilla/5.0" http://data1.f3322.net:666/share/kitti/data_object_velodyne.zip &，让服务器在后台下载，本地terminal可以关掉。设置了断点重传-c，本次下载不成功，下次也可以继续下载。
