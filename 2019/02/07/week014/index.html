Two Sum

  1. 指定target，输入数组无序
  2. 返回index
  3. 无重复数，答案唯一
  4. 用hash map(unordered map)，时间复杂度O(n)
    用map，key=数，value=index。当然unordered_map<int,int>::iterator可以简写auto：

3Sum

  1. target==0，输入数组无序
  2. 返回value
  3. 有重复数，需要去重
  4. 两层循环，时间复杂度O(n^2)
