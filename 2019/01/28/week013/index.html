Week13总结(1.28——2.3)

每周小结


下周一就是除夕啦，一年中最快乐的一天。心安理得地什么也不用干。
有点想回家了。昨天看我家那闺女傅园慧好棒啊，她在欧洲训练，非常非常辛苦的时候感到崩溃，“我就一直在想，为什么我会这么的难过。后面我才意识到，是因为我犹豫不决，不够坚定。又想放弃，又想坚持。你要做任何事情，一定要坚定才能做得好。”
现在我的也是这样呀，又想回家歇着，又觉得应该学习。
所以，最后我决定，要坚定要坚持，我还有好多东西没看，回家肯定看不了，春节一周都可以完全休息。最后的几天，要继续向前啊。毕竟还有63天四月就到了。

  • Best Time to Buy and Sell Stock with Cooldown 这个题好好啊，喜欢。因为cooldown的时间是固定的，可以用O(1)或者O(n)的space。
  • Best Time to Buy and Sell Stock II 这道题目反对也特别多。就是上一道题去除cooldown。还是一次只能持有一个stock。

  • Sort Colors

Delete Node in a Linked List
