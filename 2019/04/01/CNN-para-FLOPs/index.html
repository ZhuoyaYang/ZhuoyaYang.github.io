CNN相关

发表于 2019-04-01 | 分类于 CNN | 阅读次数:


关于CNN中的参数量、计算量。

  • 输入大小: $W \times H \times C $
  • 卷积大小: $W_{k} \times H_{k} \times C \times N$，其中，$C$为input channel，$N$为output channel。
  • 卷积后的feature map大小: $w’ \times h’ \times N$

卷积过程

CS231N-卷积过程动态演示，再结合CS231N-lecture7对卷积运算的讲解，slide-lecture7。
大小为$W \times H \times C$的input与大小为$W_{k} \times H_{k} \times C$的kernel分别进行$N$组独立的卷积运算。
对于每组运算，kernel与input对齐(depth一定是同维度的)，kernel从input内部每个位置划过，进行点积运算$W_{k}^T X + b$($X$是input的一小块区域)，得到每个位置的输出。每次滑动的计算量为$W_{k} \times H_{k} \times C + 1$。如下图所示，每次其实是进行5*5*3，75维的点乘运算。再加上偏置b，共76次运算。
重复进行这个过程，将$N$组卷积依次在input内部滑动，得到$N$组运算的输出。

参数量para

参数量只与卷积核、偏置的个数有关。
偏置的个数 = 卷积后的output channel
$$ para = N \times (W_{k} \times H_{k} \times C + 1)$$

计算量FLOPs

共有$w’ \times h’ \times N$个输出。
对于每个输出，进行了$W_{k} \times H_{k} \times C + 1$次运算，表示C个大小为$W_{k} \times H_{k}$的卷积进行卷积运算，再加上偏置。
$$ FLOPs = w’ \times h’ \times N \times (W_{k} \times H_{k} \times C + 1) = w’ \times h’ \times para$$
